The **Campus Placement Prediction System** is a full-stack machine learning application designed to predict whether a student is likely to be placed during campus recruitment.

The system follows a **three-tier microservice architecture** consisting of:

- a Node.js backend for data handling and storage,
- a React frontend for user interaction.
